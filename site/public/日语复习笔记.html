大学日语复习笔记（N5/N4）

一、高频语法速查

句型用法
〜てください请做〜。依頼・指示。「見てください。」（请看。）
〜ています正在进行 / 状态持续。「食べています。」（正在吃。）/ 「結婚しています。」（已婚状态。）
〜てもいいです可以做〜。許可。「写真を撮ってもいいです。」（可以拍照。）
〜てはいけません不可以做〜。禁止「ここでタバコを吸ってはいけません。」（这里不能吸烟。）
〜たことがあります曾经做过〜。経験。「日本に行ったことがあります。」（去过日本。）
〜たい / 〜たがる想做〜（第一人称）/ 想做〜（第三人称）。「映画が見たいです。」/「彼は行きたがっています。」
〜たり〜たりする又〜又〜（不完全列举）。「歌ったり踊ったりしました。」
〜と思います我认为〜（主观判断）。「明日は雨だと思います。」
〜つもりです打算做〜（意志/计划）。「留学するつもりです。」
〜ほうがいい最好做〜（建议）。「早く病院に行ったほうがいいです。」
〜なければならない必须做〜（义务）。「宿題をしなければなりません。」
〜なくてもいい不必做〜。免れる。「急がなくてもいいです。」（不用着急。）
〜でしょう〜吧（推测 / 确认）。「明日は寒いでしょう。」
〜から / 〜ので因为〜。原因・理由。「疲れたから/ので、もう寝ます。」
〜前に / 〜後で〜之前 / 〜之后。時間の前後。「食べる前に手を洗う。/ 食べた後で片付ける。」
〜になる / 〜くなる变得〜。変化。「暖かくなりました。」/「元気になりました。」
〜すぎる过于〜（超出限度）。「食べすぎました。」（吃太多了。）
〜ことができる能够做〜。可能。「日本語を話すことができます。」
〜ながら一边〜一边〜。並行動作。「音楽を聞きながら勉強します。」
あげる / もらう / くれる授受动词：给（外向）/ 得到 / 给（内向）。方向に注意。
より〜のほうが〜A比B更〜。比較。「電車よりバスのほうが安いです。」

三、动词变形速查

三类动词 + 六种核心变形

变形一类（五段）書く二类（一段）食べる三类 する三类 来る（くる）用法
ます形書きます食べますします来ます（きます）敬体，后续ません/ました/ましょう
て形書いて食べてして来て（きて）连接、请求、正在进行等

辞书形書く食べるする来る（くる）原形，后续〜ことが/〜前/〜つもり等
可能形書ける食べられるできる来られる（こられる）表示能力或可能性

词尾て形た形例

📋 专题一 答案与解析

1-5 B C B A B

1. まいあさ = 毎朝（每天早晨）。「毎日」读まいにち。　2. ともだち = 友達。　3. でんしゃ = 電車。　4. たてもの = 建物。　5. てんき = 天気。

6-8 A A B

6. 土曜日 = どようび（星期六）。　7. 去年 = きょねん。さくねん也是"去年"但きょねん更口语常用。　8. 料理 = りょうり（注意长音位置）。

9. 働き / しごと　两个答案都对：「働きます」（はたらきます）是动词；「仕事（しごと）をします」是用名词+する。

10. 隣 / そば / 横　「隣（となり）」指隔壁、邻近；「そば」指附近；「横（よこ）」指旁边/横向。考试中三者都可得分，最常用是「となり」。

📋 专题二 答案与解析

1. B（に）—— に 表示动作发生的具体时间点。"7点"是具体时刻，必须用に。非具体时间（今天、每天）不需要に。

2. C（で）—— で 表示动作进行的场所。図書館是"读书"这个行为发生的地点。

3. B（と）—— と 表示共同做某事的对象。"和朋友一起看电影"。

4. C（で）—— で 表示动作场所。"在这里拍照"的动作发生在"这里"。

5. D（まで）—— まで 表示移动的终点。步行的终点是"车站"。此处也可用に，但まで更强调"一直到"的过程。

6. B（が）—— が 用于可能态的能力对象。〜ことができる 前面用が标识能力的对象。

7. B（と）—— と 表示并列（全部列举）。"书和笔"。

8. C（×）—— いくら 后面不加助词，直接接动词/形容词。いくらか也可以但表"一些/某种程度"，意思不同。

9. B（より）—— より 表示比较基准。"哥哥比我高"→ Aより Bのほうが〜。

10. C（で）—— で 表示材料/手段。"蛋糕是用巧克力做的"→ N + でできる。

📋 专题三 答案与解析

1. B（食べました）—— 「昨日」是过去时间，需要用过去式（た形敬体）。

2. C（言って）—— 〜てください 固定搭配，动词必须变成て形。

3. A（していません）—— 「まだ」暗示动作尚未完成，用 〜ていない 的敬体否定。する→して＋いません。

4. C（食べたくないです）—— 想吃→食べたい；不想吃→食べたくない。

5. D（行った）—— 〜たことがある（曾经做过）固定搭配，动词用た形。

6. B（飲み）—— ながら 前接动词ます形去ます（連用形）。飲みます→飲みながら。

7. D（撮って）—— 〜てはいけません 固定搭配，动词用て形。撮る（一类：〜る→って），所以撮って。

8. C（起き）—— 〜なければなりません 前接ない形去ない。起きない→起きなければなりません。

9. 読む（よむ / 一类 -む）：て形=読んで / た形=読んだ / ない形=読まない 一类 -む词尾 → んで/んだ/まない

10. 買う（かう / 一类 -う）：て形=買って / た形=買った / ない形=買わない 一类 -う词尾 → って/った/わない

11. する（三类不规则）：て形=して / た形=した / ない形=しない

12. 来る（くる / 三类不规则）：て形=来て（きて）/ た形=来た（きた）/ ない形=来ない（こない）

📋 专题四 答案与解析

1. B（なってはいけません）—— 老师说"明天考试，不能迟到"。てはいけません = 不可以〜。

2. B（開けてもいいですか）—— 请求许可。"我可以开窗吗？"→ てもいいですか。对方回答"どうぞ"表示允许。

3. A（なる）—— つもりです 前接辞书形（原形）。"我打算将来当医生"。

4. B（したから）—— "因为昨晚学习到很晚，所以今天很困"。から 前接简体（过去用た形），因为学了→所以困。

5. C（テレビを見たり）—— 〜たり〜たり 前面两个动词都用たり形。看电视=見たり，睡觉=寝たり。

6. C（急いだ）—— 〜ほうがいい 前接た形。"最好快点"→ 急いだほうがいい。

7. D（より）—— Aは Bより 〜 = A比B更〜。"东京比大阪人口多"。

8. B（から）—— もらう 的来源用 から/に。田中さんからもらいました = 从田中那里得到。

9. B（辛）—— すぎる 前接形容词词干（去い）或动词ます形去ます。辛い→辛すぎる。"太辣了"。

10. C（×不加任何）—— 〜と思う 前面接简体句。帰ったと思う = 我觉得他已经回去了。帰っただ是错的。

📋 专题五 答案与解析

日译中：

1. 我每天早上喝完咖啡后去学校。てから：做完A之后做B，强调先后顺序。

2. 这个房间又宽敞又明亮。〜くて：形容词て形表并列，"又A又B"。

3. 昨天买的书非常有趣。買った：修饰名词时用简体（连体形）。

4. 车站前面开了一家新超市。できました：建成/出现/做好。新店开了用できる。

中译日：

5. 私は日本の寿司を食べたことがあります。注意：食べたことがある（曾经吃过），不是食べている。

6. 明日は休みですから、早く起きなくてもいいです。から 前接です/ます也可（敬体から也成立）。なくてもいい=不必〜。

7. このケーキは昨日の（ケーキ）より大きいです。比较：Aは Bより + 形容词。第二个"蛋糕"可省略。

8. 宿題をしていますから、静かにしてください。〜ています=正在进行。静かだ→静かに（副词化）+ してください。
