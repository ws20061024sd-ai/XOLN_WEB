## 列表

### 无序列表

- 这是第二项
- 这是第三项

  - 嵌套子项二
    - 更深一层（前面空四格）
    - 更深一层

* 第一项
* 第二项

## 代码

### 行内代码

在段落中可以使用 `const greeting = "你好世界"` 这样的行内代码。

常见的用法：文件名叫 `index.html`，命令用 `npm run build`。

### 代码块（带语言标注）

```javascript
// JavaScript 代码
function greet(name) {
  const message = `Hello, ${name}!`;
  console.log(message);
  return message;
}

greet("世界");
```

```python
# Python 代码
def calculate(x, y):
    result = x + y
    return result

print(calculate(10, 20))
```

```css
/* CSS 代码 */
.container {
  max-width: 680px;
  margin: 0 auto;
}
```

```html
<!-- HTML 代码 -->
<div class="card">
  <h2>卡片标题</h2>
  <p>卡片内容</p>
</div>
```

### 代码块（不带语言标注）

```
这是一段纯文本代码块。
没有语言标注，不会语法高亮。
但仍然是等宽字体。

适合放命令行输出、日志等不需要高亮的内容。
```
