## 前端连接后端

- 设置 `NEXT_PUBLIC_API_URL=http://api.xolnxoln.cn`，前端 API 调用层自动激活
- 每个详情页底部自动显示评论框和已有评论
- 提交后立即可见（无需审核）
- 关于页底部新增联系表单
- 全站自动页面访问统计（每次加载记录 PV）

## 管理面板 `/admin`

- 页脚 `©` 符号隐藏入口
- 密钥登录（默认 admin123）
- 三个 Tab：
  - **评论管理**：查看、隐藏/显示、删除评论
  - **消息查看**：查看联系表单提交的消息
  - **统计查看**：总访问量、今日访问量、各页面访问排名（中文化标注）
- 4 个管理接口（带密钥验证）

## 遇到的问题

- **CORS 错误**：后端 CORS 配置需包含 `localhost:3099`（开发环境）
- **评论不显示**：数据库旧表的 `approved` 默认值仍为 0 → INSERT 时显式写 `approved=1`
- **统计看不明白** → 加上中文标签和解释文字
