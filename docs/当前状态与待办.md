# 当前状态与待办

> 2026-05-20。根因已确认：CDN 回源用了 COS 标准端点，不支持首页索引。

---

## 一、问题根因

COS 有两个访问入口：

| 端点 | 地址 | 访问 `/` | 访问 `/index.html` |
|------|------|---------|-------------------|
| 标准端点 | `cos.ap-guangzhou.myqcloud.com` | ❌ 403 | ✅ 200 |
| 网站端点 | `cos-website.ap-guangzhou.myqcloud.com` | ✅ 自动返回 index.html | ✅ 200 |

CDN 目前配的是标准端点。当用户访问 `xolnxoln.cn/` 时：

```
用户 → xolnxoln.cn → CDN → COS标准端点请求 /
                              ↓
                         没有首页索引功能
                              ↓
                         ❌ 403 AccessDenied
```

**这就是为什么 `/index.html` 能直接访问，但根域名打不开。网站端点有首页索引功能（自动把 `/` 映射到 `index.html`），标准端点没有。**

---

## 二、解决方法

CDN 控制台 → 域名管理 → `xolnxoln.cn` → 源站配置：

1. 源站类型切为 **自有源**（COS 源类型不支持填网站端点）
2. 源站地址填 `xolnxoln-1431302682.cos-website.ap-guangzhou.myqcloud.com`
3. 回源 HOST 填同上
4. 回源协议 HTTP

`www.xolnxoln.cn` 同样操作。
