# 内容编辑指南

> 本文档随网站变化实时更新。每次网站结构或编辑方式有变动，同步更新此文档。

## 快速上手

网站所有内容保存在 `site/content/pages/` 目录下的 `.md` 文件中。改内容 = 改这些文件。

```
site/content/pages/
├── about.md        ← 个人简介
├── beliefs.md      ← 观念
├── works.md        ← 作品
├── favorites.md    ← 喜爱
├── changelog.md    ← 更新日志
└── misc.md         ← 杂项
```

### 改内容三步

1. 打开对应 `.md` 文件
2. 编辑保存
3. `git add -A && git commit -m "更新xxx"`（网站自动构建部署）

本地预览：`npm run dev` → `http://localhost:3099`

## 各栏目对应文件

| 导航名称 | 内容文件 | 功能 |
|---------|---------|------|
| 首页 | `src/app/page.tsx` | Hero 介绍 + 6 张栏目卡片，**不是 .md 文件** |
| 关于 | `content/pages/about.md` | 个人简介 |
| 观念 | `content/pages/beliefs.md` | 世界观、人生观、价值观等 |
| 作品 | `content/pages/works.md` | 文稿、设计作品 |
| 喜爱 | `content/pages/favorites.md` | 电影、音乐、书籍、活动 |
| 更新 | `content/pages/changelog.md` | 网站更新记录 |
| 杂项 | `content/pages/misc.md` | 未归类内容 |

> 首页是唯一需要改 React 代码的页面。其余栏目纯改 `.md` 文件即可。

## Markdown 写作语法

### 基本排版

```markdown
# 一级标题
## 二级标题
### 三级标题

**加粗文字**
*斜体文字*

[链接文字](https://example.com)

- 无序列表项
- 无序列表项

1. 有序列表
2. 有序列表

> 引用文字

---   ← 分割线

`行内代码`
```

### 代码块

````markdown
```python
def hello():
    print("Hello World")
```
````

### 表格

```markdown
| 列A | 列B | 列C |
|-----|-----|-----|
| 数据 | 数据 | 数据 |
| 数据 | 数据 | 数据 |
```

### 图片

```markdown
![图片描述](图片地址)
```

**本地图片**：放 `public/images/`，引用路径 `/images/文件名.jpg`

```
public/images/photo.jpg  →  ![我的照片](/images/photo.jpg)
```

**远程图片**：直接写完整 URL

```markdown
![示例](https://example.com/image.jpg)
```

图片自动懒加载，不影响页面加载速度。

### 视频

**本地视频**（放 `public/videos/`）：

```markdown
!video[标题](/videos/demo.mp4)
```

**Bilibili**（复制视频的 BV 号）：

```markdown
!bilibili[标题](BV1xx411c7mD)
```

**YouTube**（复制视频 ID）：

```markdown
!youtube[标题](dQw4w9WgXcQ)
```

视频以 16:9 响应式容器展示，支持全屏播放。

## 常见操作

### 改首页自我介绍

编辑 `src/app/page.tsx`，找到并修改：

```tsx
// 第 55 行附近
<span className="...">[你的名字]</span>   // ← 改成你的名字

// 第 62-63 行附近
<p className="...">
  这里是我的数字花园。记录思考，展示作品，整理一切塑造了我的事物。
  // ↑ 改成你的介绍文字
</p>
```

### 改网站标题

编辑 `src/app/layout.tsx`：

```tsx
export const metadata: Metadata = {
  title: "我的网站",   // ← 浏览器标签上显示的标题
};
```

### 改导航栏标签

编辑 `src/components/Header.tsx`，修改 `navItems` 数组：

```tsx
const navItems = [
  { href: "/", label: "首页" },
  { href: "/about", label: "关于" },    // ← 改成你想要的名字
  // ...
];
```

### 添加新栏目

1. 在 `content/pages/` 新建 `.md` 文件
2. 在 `src/app/` 新建目录 + `page.tsx`
3. 在 `Header.tsx` 的 `navItems` 中加入新条目
4. 在 `src/app/page.tsx` 的 `sections` 数组中添加卡片

### 修改配色/字体

编辑 `src/app/globals.css`，找到 `:root` 块：

```css
:root {
  --accent: #2563eb;        /* 强调色（链接、按钮） */
  --bg: #fafaf8;            /* 页面背景色 */
  --text: #1a1a1a;          /* 文字颜色 */
  --text-muted: #6b6b6b;    /* 辅助文字 */
}
```

改颜色值即可，全站自动更新。

### 修改首页卡片图标

编辑 `src/app/page.tsx`，每个 `section` 对象的 `icon` 字段是 SVG。可以用 [Lucide Icons](https://lucide.dev) 找图标，复制 SVG 代码替换。

## 图片与视频资源目录

```
public/
├── images/           ← 所有图片
│   └── photo.jpg
├── videos/           ← 所有视频
```

> `public/` 下的文件在构建时原样复制到网站根目录。`public/photo.jpg` → 可通过 `/photo.jpg` 访问。
